# SafeSend
Helping victims of domestic violence, trafficking, or situations of immediate danger to send a distress signal without arousing the suspicions of the abuser.

It's an Android application developed with a .dart extension that disguises itself as a regular app (calculator, weather app, or shopping list). The user configures a specific gesture (e.g., entering a secret code on the fake calculator, or shaking the phone in a certain way) which, without changing the display, triggers a chain of secure actions.
